Python का पूरा परिचय

Python क्या है?

Python एक उच्च-स्तरीय, व्याख्या योग्य प्रोग्रामिंग भाषा है, जिसे 1991 में Guido van Rossum ने बनाया था। यह भाषा सरल, पठनीय और बहुमुखी है, जिसका उपयोग वेब डेवलपमेंट, डेटा साइंस, मशीन लर्निंग, ऑटोमेशन और बहुत कुछ में किया जाता है। Python का सिद्धांत "कोड को पढ़ना आसान होना चाहिए" है, इसलिए यह शुरुआती लोगों के लिए आदर्श है।

Python की विशेषताएं

  • सरल सिंटैक्स: Python का कोड अन्य भाषाओं जैसे C++ या Java की तुलना में कम जटिल है।
  • व्याख्या योग्य: इसे चलाने के लिए कंपाइलर की आवश्यकता नहीं होती; यह लाइन-बाय-लाइन चलती है।
  • लाइब्रेरीज़: NumPy, Pandas, TensorFlow जैसी शक्तिशाली लाइब्रेरीज़ उपलब्ध हैं।
  • क्रॉस-प्लेटफॉर्म: Windows, macOS, Linux पर चलती है।
  • ओपन-सोर्स: मुफ्त और समुदाय द्वारा समर्थित।

बेसिक ट्यूटोरियल्स

1. वेरिएबल्स और डेटा टाइप्स

Python में वेरिएबल्स डेटा स्टोर करने के लिए उपयोग होते हैं। उदाहरण:

name = "John"
age = 25
height = 5.9
is_student = True

डेटा टाइप्स: str (string), int (integer), float, bool।

2. कंडीशनल स्टेटमेंट्स

if-else का उपयोग निर्णय लेने के लिए:

if age > 18:
    print("Adult")
else:
    print("Minor")

उन्नत ट्यूटोरियल्स

1. फाइल हैंडलिंग

फाइल पढ़ना/लिखना:

with open("file.txt", "w") as f:
    f.write("Hello!")

with open("file.txt", "r") as f:
    print(f.read())

2. OOP (Object-Oriented Programming)

क्लासेस और ऑब्जेक्ट्स:

class Dog:
    def __init__(self, name):
        self.name = name
    def bark(self):
        print("Woof!")

dog = Dog("Buddy")
dog.bark()
